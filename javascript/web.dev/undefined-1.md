# 사용하지 않는 코드 제거

* npm과 같은 레지스트리 => 누구나 쉽게 다운로드하여 50만 개 이상의 공개 패키지를 사용할 수 있게 하여&#x20;
  * Javascript 세계를 더 나은 방향으로 변화시킴&#x20;
*   우리는 종종 완전히 활용하지 않는 라이브러리를 포함함&#x20;

    * 번들을 분석하여 사용되지 않는 코드를 감지
    * 사용하지 않고 필요하지 않은 라이브러리를 삭제해야 함&#x20;
