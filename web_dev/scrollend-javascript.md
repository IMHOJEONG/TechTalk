# Scrollend, 새로운 Javascript 이벤트

* scrollend 이벤트 이전엔 스크롤이 완료되었는지 감지할 수 있는 신뢰할 수 있는 방법이 없었음&#x20;
* 즉, 이벤트가 늦게 시작되거나 사용자의 손가락이 화면에 계속 내려져 있는 동안 발생&#x20;
* 스크롤이 실제로 종료된 시점을 알 수 없기 때문에 버그가 발생하고 사용
